# Running the project
Clone the project
`git clone git@github.com:giovanedann/gpt3-project.git`

Choose the directory
`cd gpt3-project`

Install the dependencies and wait for the installation
`npm install`

Start the project
`npm start`

Access the project on https://localhost:3000
